HR Analytics SQL Project

Project Overview:-
- This project focuses on analyzing HR data using SQL to extract meaningful business insights related to employee performance, salary distribution, departmental trends, and workforce analytics.
- The objective of this project is to demonstrate strong SQL querying skills, including data aggregation, filtering, grouping, sorting, and analytical problem-solving.

Business Objective:-
- To analyze employee data and answer key HR-related business questions such as:
1. Which department has the highest average salary?
2. What is the distribution of employees across departments?
3. Who are the top-performing employees?
4. What is the average experience level by department?
5. How does salary vary with experience?

Tools & Technologies:-
1. SQL
2. MySQL
3. Relational Database Concepts
4. Dataset Description

The dataset includes the following attributes:-
1. Employee_ID
2. Employee_Name
3. Department
4. Salary
5. Experience_Years
6. Performance_Rating
7. Joining_Date

Key SQL Concepts Used:-
1. SELECT statements
2. WHERE clause filtering
3. GROUP BY and HAVING
4. Aggregate functions (AVG, COUNT, SUM, MAX)
5. ORDER BY
6. CONCAT function
7. Subqueries

Key Insights Generated:-
1. Identified departments with the highest average salary
2. Determined the top 5 highest-paid employees
3. Analyzed average performance rating by department
4. Evaluated salary trends based on experience levels
5. Learning Outcome

This project strengthened my ability to:-
- Translate business questions into structured SQL queries.
- Perform analytical data extraction using relational databases.
- Generate actionable insights from structured datasets.

Conclusion:-
- This project demonstrates the practical application of SQL in solving real-world HR business problems. By transforming raw employee data into structured insights, the analysis supports   data-driven decision-making related to compensation strategy, performance evaluation, and workforce planning.
- Through this project, I showcased strong analytical thinking, structured query writing, and the ability to connect technical outputs with business objectives.
